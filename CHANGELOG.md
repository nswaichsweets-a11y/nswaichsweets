# Changelog

## 0.1.0 - Initial agency showcase build

- Added Next.js App Router PWA foundation.
- Added Prisma PostgreSQL schema for users, roles, permissions, business settings, products, invoices, payments, ledgers, inventory, milk collection, employees, attendance, salaries, expenses, report exports, and audit logs.
- Added role-based users with zero business data for clean production setup.
- Added signed cookie authentication and protected app shell.
- Added responsive desktop sidebar and mobile bottom navigation.
- Added dashboard metrics, module pages, forms, reports catalogue, PDF exports, and CSV exports.
- Added deployment, database, backup, performance, and change-management documentation.
- Added Gmail login hooks, forgot-password reset links, super-admin access controls, POS order printing, and clean-data safeguards.
- Added sweets-shop menu departments, supplier contact backup fields, separate supplier ledger page, business WhatsApp setting, and WhatsApp invoice sharing with public invoice links.
- Added farmer profile placeholders, daily milk entry selection, packaging/disposable inventory types, compulsory employee Aadhaar field, invoice assigned-name stamping, super-admin-only business identity settings, and non-tax invoice title.
